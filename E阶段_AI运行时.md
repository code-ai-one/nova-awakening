### E.1.2 · 类人脑架构研究
**生物神经元特性**:
- 脉冲触发（all-or-nothing）
- 时序依赖（STDP · spike-timing-dependent plasticity）
- 稀疏激活（<5%神经元同时活跃）
- 局部连接（皮层柱结构）
- 多尺度记忆（海马体→皮层）

### E.1.3 · 原创神经元模型 · 动态阈值脉冲神经元 (Dynamic-Threshold Spike Neuron, DTSN)
```
状态: (膜电位 V, 阈值 θ, 适应因子 a)
输入: 加权脉冲 s_i(t) * w_i
更新:
  V(t+1) = α·V(t) + Σᵢ w_i·s_i(t) - a·(V-V_静)
  若 V(t) ≥ θ, 则:
    发放脉冲 s_out(t) = 1
    V(t+1) = V_重置
    θ(t+1) = θ + Δθ    // 阈值自适应
    a(t+1) = a · ρ     // 适应增强
  否则:
    θ(t+1) = θ - decay·(θ - θ_基)
    a(t+1) = a · decay_a
```

**创新**: 
- 动态阈值使神经元可以"累积"和"重置"，比LIF模型更灵活
- 适应因子实现类生物的"疲劳" + "兴奋"机制
- 纯二元脉冲输出 → 稀疏 → 能耗低

### E.1.4 · 原创学习规则 · 效用导向STDP (Utility-STDP)
```
经典STDP: Δw_ij = η·f(Δt_ij)  // 只依赖时间差
Utility-STDP (自研):
  Δw_ij = η · f(Δt_ij) · (1 + γ · U_global)
  其中 U_global 是全网效用信号（基于任务奖励或无监督目标）
```

**创新**: 将全局效用信号注入局部STDP规则，兼具局部性+全局导向。

### E.1.5 · 原创激活函数 · 脉冲序列代数 (Spike-Train Algebra)
```
定义 脉冲序列 s ∈ {0,1}^T
运算:
  s₁ ⊕ s₂ = 逐位OR (突触累积)
  s₁ ⊗ s₂ = 时间延迟卷积 (STDP)
  s₁ ⊙ τ = 频率调制 (时间尺度变换)
```

### E.1.6 · 原创注意力 · 局部竞争注意力 (LCA · Local Competition Attention)
```
传统 self-attention: O(n²) 
LCA (自研): O(n · log n) · 基于K-邻近 + 侧抑制

算法:
  1. 每个token与其K个邻居（可学习位置）交换信息
  2. 邻居内部竞争，只有最强的通过（top-1）
  3. 多层堆叠实现长距离感受野
```

### E.1.7 · 原创位置编码 · 螺旋位置码 (Spiral Position Code)
```
传统 sin/cos:  
  PE(pos, 2i) = sin(pos/10000^(2i/d))
  PE(pos, 2i+1) = cos(pos/10000^(2i/d))

螺旋码 (自研):
  极坐标 (r, θ):
    r(pos) = sqrt(pos + 1)           // 距离原点距离
    θ(pos) = pos · golden_angle      // 黄金角度
  转为 d维向量:
    PE(pos) = 投影(r, θ, d)
```

**创新**: 模仿向日葵种子/海螺壳的自然螺旋，每个位置与所有其他位置有唯一几何关系。

### E.1.8 · 原创稀疏性机制 · 竞争性top-K
```
Softmax (dense): 所有神经元都有梯度
Top-K Competitive:
  激活 = top_K(预激活) 
  其余置为0
  反向传播只更新top-K神经元 → 稀疏梯度 → 更快训练
```

### E.1.9 · 原创记忆机制 · 双路海马-皮层架构
```
短期记忆 (海马): 快速写入·容量小·衰减快
长期记忆 (皮层): 慢速巩固·容量大·持久

巩固过程:
  重要事件 → 海马 → 睡眠期迁移 → 皮层
  检索: 
    先查海马（快）
    海马无结果 → 皮层模式完成
```

```
挑战: 脉冲神经元不可微（阶跃函数）

  前向: spike = (V ≥ θ) ? 1 : 0
```

## 独创性总结

### 全球首创
1. **动态阈值脉冲神经元DTSN** · 结合阈值适应+疲劳恢复
2. **效用导向STDP** · 全局+局部学习信号融合
3. **脉冲序列代数** · 将脉冲计算抽象为代数结构

6. **双路海马-皮层记忆** · 短期快速+长期巩固

### 对比transformer
| 指标 | Transformer | Nova DTSN |
|------|------------|-----------|
| 注意力复杂度 | O(n²) | O(n log n) |
| 激活稀疏度 | ~100% | <5% |
| 能耗 | 高(稠密FP32) | 低(稀疏二值) |
| 长期记忆 | 无 | 海马-皮层 |
| 可解释性 | 黑盒 | 脉冲可视化 |
