## 🔴 P0 · 真实Bug（影响正确性）

### Bug-1: 数据流分析未声明变量`已在` 
**文件**: `@/home/cch/桌面/新觉醒/原生编译器/编译器/分析/数据流分析.nova:220-225,289-294`

**现象**:
```nova
// 数流_前向求解 L216-225
定义 可变 j = 0
当 (j < n) {
    如果 (数流_块_ID(块列表[j]) == 后继ID) {
        定义 可变 k = 0
        当 (k < 计数(工作列表)) {
            如果 (工作列表[k] == j) { 已在 = 真 }  // ❌ 已在未声明!
            k = k + 1
        }
        如果 (已在 == 假) { 追加(工作列表, j) }  // ❌ 读取未初始化!
    }
    j = j + 1
}
```

**根因**: 内层循环忘记在开始处声明`定义 可变 已在 = 假`

**修复**（两处）:
```nova
// 在 j循环开始处添加
定义 可变 j = 0
当 (j < n) {
    如果 (数流_块_ID(块列表[j]) == 后继ID) {
        定义 可变 已在 = 假      // ✅ 补上
        定义 可变 k = 0
        ...
```

**影响**: 数据流分析在工作列表中查找已有块时行为未定义，可能导致worklist无限循环或漏加后继。

## 🟡 P1 · 重复逻辑（可重构）

### Dup-1: 注释提取待办/待修 几乎完全相同
**文件**: `@/home/cch/桌面/新觉醒/原生编译器/编译器/词法/注释处理.nova:175-213`

**现象**: `注释_提取待办`(TODO)和`注释_提取待修`(FIXME)两个函数**只有关键字和长度不同**，其余代码**逐字相同**。

**重构方案**:
```nova
// 提取公共函数
函数 注释_提取标记(源码, 标记, 标记长, 类型名) {
    定义 可变 结果 = []
    定义 可变 位置 = 0
    定义 长 = 计数(源码)
    定义 可变 行号 = 1
    当 (位置 < 长) {
        如果 (位置 + 标记长 < 长 且 子串(源码, 位置, 标记长) == 标记) {
            定义 可变 内容起始 = 位置 + 标记长
            如果 (内容起始 < 长 且 编码(子串(源码, 内容起始, 1)) == 58) { 内容起始 = 内容起始 + 1 }
            当 (内容起始 < 长 且 编码(子串(源码, 内容起始, 1)) == 字符_空格) { 内容起始 = 内容起始 + 1 }
            定义 可变 内容结束 = 内容起始
            当 (内容结束 < 长 且 编码(子串(源码, 内容结束, 1)) != 10) { 内容结束 = 内容结束 + 1 }
            追加(结果, [行号, 类型名, 子串(源码, 内容起始, 内容结束 - 内容起始)])
        }
        如果 (编码(子串(源码, 位置, 1)) == 字符_换行) { 行号 = 行号 + 1 }
        位置 = 位置 + 1
    }
    返回 结果
}

// 原函数改为wrapper
函数 注释_提取待办(源码) { 返回 注释_提取标记(源码, "TODO", 4, "待办") }
函数 注释_提取待修(源码) { 返回 注释_提取标记(源码, "FIXME", 5, "待修") }
```

**收益**: 减少70%重复代码，修bug只需改一处。

### Dup-2: 修正表 取类型/取位置/取附加 同构
**文件**: `@/home/cch/桌面/新觉醒/原生编译器/编译器/规划/核心/常量.nova:656-682`

**现象**: 三个函数结构完全相同：
```nova
函数 修正表_取类型(表, 索引) {
    如果 (是列表(表) == 假 或 索引 < 0 或 索引 >= 计数(表)) { 返回 空 }
    如果 (类型(表[索引]) == "列表") { ... 返回 表[索引][0] }
    返回 表[索引]
}
// 取位置 - 偏移1, 取附加 - 偏移2, 逻辑完全同构
```

**重构方案**: 
```nova
函数 修正表_取字段(表, 索引, 字段偏移) {
    如果 (是列表(表) == 假 或 索引 < 0 或 索引 >= 计数(表)) { 返回 空 }
    如果 (类型(表[索引]) == "列表") {
        如果 (计数(表[索引]) > 字段偏移) { 返回 表[索引][字段偏移] }
        返回 空
    }
    定义 实际索引 = 索引 + 字段偏移
    如果 (实际索引 >= 计数(表)) { 返回 空 }
    返回 表[实际索引]
}

// 保留原API作为薄wrapper
函数 修正表_取类型(表, 索引) { 返回 修正表_取字段(表, 索引, 0) }
函数 修正表_取位置(表, 索引) { 返回 修正表_取字段(表, 索引, 1) }
函数 修正表_取附加(表, 索引) { 返回 修正表_取字段(表, 索引, 2) }
```
