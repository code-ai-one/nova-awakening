## C.1 原创IR代数 · 意图表示

### 核心公式 · 意图代数 IA (Intent Algebra)
```
定义 IA 为三元组 (V, E, ⊙)
其中:
  V = 值集 (Value)          // 常量、变量、表达式求值结果
  E = 效应集 (Effect)        // 读/写/异常/IO/随机
  ⊙: V × V → V + E         // 值域上的代数运算, 可能产生效应
```

### 意图态 (Intent State)
每个IR节点是一个**意图态** `τ = ⟨值, 效应集, 来源⟩`:
- `值` ∈ V (或 ⊥ 表示未知)
- `效应集` ⊆ E (此节点的副作用)
- `来源` ∈ 源码位置

### 意图等价关系 ≡
```
τ₁ ≡ τ₂  当且仅当
  值(τ₁) = 值(τ₂) 且 效应集(τ₁) = 效应集(τ₂)
(来源可以不同, 保持调试信息)
```

### IA优化 = 代数同构变换
```
Opt: IR → IR 是保意图的 当且仅当
  ∀节点 n: Opt(n) ≡ n
```

### 意图范畴 Cat(IA)
- 对象: 类型 T ∈ Type
- 态射: τ: T₁ → T₂ (纯函数或带效应的变换)
- 组合: τ₂ ∘ τ₁ (管道组合)
- 恒等: id_T = (x: T) ↦ x

## C.2 原创优化算法

### C.2.1 · 常量折叠·代数证明
```
命题: 若 τ = n₁ ⊙ n₂, 且 值(n₁)=v₁, 值(n₂)=v₂ 均已知,
     则 τ 可替换为 ⟨v₁⊙v₂, 效应(n₁)∪效应(n₂), τ.来源⟩
证明: 值域封闭性 + 效应保持
```

### C.2.2 · 死代码消除·数学证明
```
命题: 若 τ 的效应集 = ∅ 且 τ 不被任何其他节点依赖,
     则删除 τ 保语义
证明: 纯粹性 + 无引用 → 不影响观察等价
```

### C.2.3 · 公共子表达式消除 CSE
```
算法: 
  1. 构建哈希索引: τ 的规范形 → 节点ID
  2. 遇到新 τ, 若规范形已存在, 复用旧ID
  3. 修正引用链
```

### C.2.4 · 循环不变量外提 LICM
```
定义 不变量(e, L) 当且仅当 e 在循环 L 中求值恒不变
算法: 
  1. 支配树分析
  2. 识别循环头
  3. 对每个循环体节点, 检查所有输入是否为不变量
  4. 外提到循环前置头
```

## C.3 原创代码生成

### C.3.1 · 图着色+线性扫描混合寄存器分配
```
算法 Nova-RA:
  1. 构建冲突图 G=(V,E), V=变量, E=同时活跃
  2. 初始着色: 线性扫描 (O(n log n))
  3. 溢出优化: 对颜色冲突, 用图着色调整
  4. 最小化溢出: 按使用频率权重
```

### C.3.2 · 树模式匹配+DP指令选择
```
算法:
  1. IR树 分解为 "瓦片(tile)"
  2. 每个瓦片匹配 目标指令模式
  3. DP选择最小代价的瓦片集合
  代价 = 指令周期 + 内存访问 + 依赖链
```

### C.3.3 · 数据流图调度
```
输入: 基本块B的指令序列
输出: 重排后的序列, 最小化流水线stall

算法:
  1. 构建数据依赖图 DDG
  2. 拓扑排序 + 就绪队列
  3. 每个时钟周期选择就绪指令:
     - 优先级: 关键路径 > 资源冲突 > 依赖距离
```

## C.5 模块系统创新

### C.5.1 · 原创模块图算法
```
Nova 模块图:
  节点: 模块 M = (源路径, 导入集, 导出集, 哈希)
  边: M₁ → M₂ 表示 M₁ 导入 M₂
  
算法:
  1. 拓扑排序 (Kahn算法) - O(V+E)
  2. 循环检测 (Tarjan强连通分量) 
  3. 并行编译 (无依赖的同层)
```

### C.5.2 · 模块签名（编译期契约）
```
签名 σ(M) = (导出函数签名, 导出类型, 效应标记)
验证: 每次编译后, σ(M_新) ⊇ σ(M_旧) 保证向后兼容
```

### 关键定理
```
定理 1 (编译器正确性):
  ∀源码 s, 编译(s) 生成的机器码 m 满足:
    执行(m) ≡ 执行(解释器(s))

定理 2 (自举一致性):
  ∀n≥2: 编译器_stage_n = 编译器_stage_(n+1)

定理 3 (优化保语义):
  ∀pass P, ∀程序 p:
    执行(P(p)) ≡ 执行(p)
```
